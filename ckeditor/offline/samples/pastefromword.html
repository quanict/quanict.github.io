Pasting content from Microsoft Word Documentation

The Paste from Word feature is provided through the Paste from Word plugin. It was heavily reworked in CKEditor 4.6 and by default is available in the Standard and Full distributions.

This feature allows you to paste content from Microsoft Word and maintain original structure and formatting. After creating a document in Word you can simply copy it to CKEditor and retain font styles, heading levels, nested lists, complex tables and other advanced features.

When the plugin is enabled, it automatically detects Word content and transforms its structure and formatting to clean HTML. It also adds the   toolbar button which makes it possible to paste clipboard data this way only on demand.

To test how Paste from Word works, download the sample Word document, open it in Microsoft Word, copy the content and paste it into CKEditor. Check the clean HTML source code generated by CKEditor using the Source button. To test even more styles and formatting, download another sample Word document with more complicated structure and styling and try to paste its content into CKEditor.

Pasted content may be filtered by additional Advanced Content Filter rules. The content appearance in CKEditor also depends on styling used in contents.css files.
